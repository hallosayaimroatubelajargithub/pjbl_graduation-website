# Website Pengumuman Kelulusan Kelas XII
Website Pengumuman Kelulusan merupakan aplikasi web yang dirancang untuk menampilkan hasil kelulusan murid secara digital, cepat, dan mudah diakses. Pengguna dapat masuk menggunakan kode autentik untuk melihat status kelulusannya secara aman dan sederhana.

Projek ini dikembangkan sebagai media pembelajaran berbasis **Project Based Learning (PjBL)** yang mengintegrasikan konsep algoritma, _flowchart, wireframe_, _UI/UX design_, dan pengembangan website. Melalui projek ini, murid tidak hanya belajar menggunakan teknologi, tetapi juga memahami proses perancangan dan pengembangan sistem digital dari kebutuhan pengguna hingga produk akhir.

**Fitur Utama**
1. Halaman beranda yang informatif
2. Login menggunakan kode autentik
3. Validasi akses pengguna
4. Tampilan hasil kelulusan (Lulus/Tidak Lulus)
5. Antarmuka (UI) yang sederhana dan mudah digunakan
6. Responsif dan mudah diakses oleh siswa

**Tujuan Proyek** </br>
Membantu murid memahami penerapan nyata algoritma dan pemrograman dalam pengembangan website, sekaligus menyediakan sistem pengumuman kelulusan yang lebih modern, efisien, dan ramah pengguna.
